# zone-sdk-test / yolo-board (UI plugin)

> **Moved.** This UI plugin is now `ui/` inside the [`vpavlin/logos-yolo`](https://github.com/vpavlin/logos-yolo) monorepo.
>
> This repository is kept read-only for history. All new work — issues, PRs, commits — happens in the monorepo.

The QML plugin (`yolo_board`) is a thin client: it calls into the `yolo_board_module` backend (formerly `logos-yolo-board-module`, now `module/` in the same monorepo) over QRO IPC and renders channels, messages, media, and per-message threads.

## Quickstart (new location)

```bash
git clone https://github.com/vpavlin/logos-yolo
cd logos-yolo
# follow the README there
```
